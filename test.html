⚡ CORS Preflight 회피 테스트

POST 요청 방식을 바꿔서 preflight 없이 테스트
💡 GET 테스트가 성공한 같은 URL을 사용합니다
💡 CORS Preflight 회피 전략 문제: POST + Content-Type: application/json → preflight 요청 발생 해결: 다른 Content-Type 사용 → preflight 회피 테스트할 방법들: 1. FormData 사용 2. Plain Text 사용 3. URL Encoded 사용 모든 방법이 실패하면 Google Apps Script에 doOptions 함수를 추가해야 합니다.
